## The Galery

is a simple galery website use vue3 and tailwindcss3

how to install

1. clone the repository to you local comouter
```
  git clone https://github.com/ka-shifuka/the-galery
```

2. install dependencies use npm
```
  npm install
```
or use yarn
```
  yarn install
```
or use pnpm (i recomended)
```
  pnpm install
```

3. run the project
```
  npm run dev
  yarn run dev
  pnpm run dev
```
